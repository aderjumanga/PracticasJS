JavaScript desde cero clase-01

Insertar JavaScript dentro de HTML

Se usa la etiqueta <scripts> </script> y se puede escribrir código javascript dentro de ella, o dejarla vacia y llamar a un atributo .js externo usando el atributo src.

Sintaxis de JavaScripts

  • Case sensitive (es sensible a las mayúsculas y minuscúlas)
  • Es de tipado dinámico
  • Comentarios de en JS se pueden escribir de dos maneras:
    • /* */ (Multilineal)
    • // (para cada liena, recomendada)
  • Todas las sentencias deben terminar con punto y coma
  • Por regla general, despues de las llaves no deben ir punto y coma, salvo dos exepciones:
    • Cuando se crea un objeto literal.
    • Cuando se crea una funcion con expresiones
